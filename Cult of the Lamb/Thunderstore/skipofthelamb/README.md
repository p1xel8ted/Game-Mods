# Skip of the Lamb

A mod to skip various cutscenes and intros in Cult of the Lamb.

## Features

### Pre-Rendered Videos
- **Skip Splash Screens** - Skip developer/publisher logo videos on startup
- **Skip Crown Video** - Skip the video when the player first dies and is crowned by the death cat
- **Skip Woolhaven Video** - Skip the pre-rendered video after completing the Woolhaven intro dungeon

### In-Game Cutscenes
- **Skip Woolhaven Arrival** - Skip the in-game cutscene when first arriving in Woolhaven
- **Skip Main Boss Intros** - Skip in-game boss ritual/transformation sequences
- **Skip Mini-Boss Intros** - Skip in-game mini-boss roar/spawn sequences
- **Ignore First Encounter** - Skip boss intros even on first encounter (useful for veteran players on new saves)

## Just Want Splash Screens Skipped?

Check out [Skip of the Lamb Lite](https://cult-of-the-lamb.thunderstore.io/package/p1xel8ted/Skip_of_the_Lamb_Lite/) - a lightweight version with no configuration required.

## Compatibility

- **Skip of the Lamb Lite** - Incompatible. Only one variant can be loaded at a time.
- **Cult of QoL** - Fully compatible. Skip features have been removed from CultOfQoL.

## Configuration

All features are disabled by default. Edit the config file in `BepInEx/config/` or install [BepInEx Configuration Manager](https://cult-of-the-lamb.thunderstore.io/package/p1xel8ted/BepInEx_Configuration_Manager/) to configure in-game with F1.
