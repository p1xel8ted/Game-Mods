# Skip of the Lamb Lite

A lightweight mod that skips developer/publisher splash screens on game startup. No configuration required - just install and play.

## Features

- **Skip Splash Screens** - Automatically skips developer/publisher logo videos when launching the game

## Installation

* Install [BepInExPack CultOfTheLamb](https://thunderstore.io/c/cult-of-the-lamb/p/BepInEx/BepInExPack_CultOfTheLamb/)
* Place the plugin DLL into your "...\Cult of the Lamb\BepInEx\plugins" folder.

## Compatibility

- **Skip of the Lamb** - Incompatible. Only one variant can be loaded at a time.
- **Cult of QoL** - Fully compatible.

## Looking for More?

If you want additional skip features (crown video, Woolhaven cutscenes, boss intros), check out the full [Skip of the Lamb](https://thunderstore.io/c/cult-of-the-lamb/p/p1xel8ted/Skip_of_the_Lamb/) mod instead.
