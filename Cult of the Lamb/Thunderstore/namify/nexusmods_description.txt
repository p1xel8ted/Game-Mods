[size=4][b]Features[/b][/size]

[list]
[*][size=3]Gets a random list of 1000 names from the randommer.io API for use when naming followers.[/size]
[*][size=3]Names are pre-loaded on game start for instant availability during indoctrination.[/size]
[*][size=3]The name list is saved locally to prevent having to fetch names every time you start the game.[/size]
[*][size=3]Names are removed from the list as they are used to prevent duplicates.[/size]
[*][size=3]The provided API key is limited to 1000 requests per day. If you want to use your own API key, you can get one from [url=https://randommer.io/]randommer.io[/url].[/size]
[*][size=3]Add your own names to the list by editing the JSON file or by using the interface via F1. The file is saved unencrypted to allow users to edit the file themselves.[/size]
[*][size=3]Generated names display an asterisk during indoctrination so you can tell them apart from your own names. The asterisk is removed automatically once the name is confirmed. This can be toggled via F1.[/size]
[/list]

[size=4][b]Installation[/b][/size]

[list]
[*][size=3]Install [url=https://www.nexusmods.com/cultofthelamb/mods/31]Cult of the Lamb BepInEx 5 Pack[/url][/size]
[*][size=3]Install [url=https://thunderstore.io/c/cult-of-the-lamb/p/p1xel8ted/BepInEx_Configuration_Manager/]BepInEx Configuration Manager[/url] (for in-game configuration)[/size]
[*][size=3]Place the plugin DLLs into your "...\Cult of the Lamb\BepInEx\plugins" folder.[/size]
[/list]

[size=4][b]Configuration[/b][/size]

[list]
[*][size=3]Press F1 in game to open the configuration manager.[/size]
[*][size=3]The configuration file can be found in the "...\Cult of the Lamb\BepInEx\config" folder.[/size]
[/list]
