## Features

### Menu & UI
- Remove menu clutter (Discord, bug report, Twitch buttons)
- Remove pause menu buttons (Help, Twitch, Photo Mode)
- Disable main menu ads and logos
- Disable main menu dark mode glitch effect
- Hide New Game button when saves exist
- Disable all notifications (with option to allow critical ones like deaths)
- Customise notification scale

### Saves
- Auto-load into a specific save slot on game start
- Save on quit to desktop/menu
- Quick save keybind

### Gameplay
- Easy fishing (auto-reel, no button mashing)
- Disable game over mechanic
- Fast rituals and sermons
- Reverse Ritual of Enrichment nerf (coins scale with follower level)
- Customise ritual cooldowns

### Combat & Movement
- Modify run, dodge, and lunge speeds
- Disable speed boosts in dungeons or combat
- Modify base damage multiplier
- Reverse Golden Fleece damage cap (patch 1.0.13)
- Customise Golden Fleece damage multiplier

### Game Speed
- Manipulate game speed with keybinds (up to 5x)
- Shorten speed increments (1 instead of 0.25)
- Slow down time (make days longer)

### Tarot Cards
- 3x rare card draw chance
- Rare cards only mode

### Loot
- Magnetize all loot to player
- Customise magnet range multiplier

### Followers
- Cleanse illness/exhaustion on level up
- Uncap follower level benefits beyond level 10
- Collect tithe from elderly followers
- Intimidate elderly followers
- Make elderly followers work
- Replace follower necklaces (old one drops to ground)
- Customise life expectancy range
- Prioritize followers with active requests in selection lists
- Auto-select best mating pair in Mating Tent

### Mass Actions
- Mass inspire, bless, bribe, extort, intimidate
- Mass fertilize, water
- Mass level up
- Mass bully, reassure, reeducate
- Mass romance, pet dog
- Mass pet farm animals

### Mass Collect
- Mass collect from beds, outhouses, offering shrines
- Mass collect from passive shrines, compost bins, harvest totems

### Structures
- Customise Harvest Totem radius
- Customise Propaganda Speaker radius (and disable audio/night mode)
- Customise Farm Station radius
- Customise Farm Plot Sign radius
- Heal exhausted followers in Healing Bay
- Hide healthy followers from Healing Bay
- Show only dissenters in Prison

### Resource Production
- Lumber/stone mines: extend lifespan (50%, 100%, or unlimited)
- Wood silos produce spider webs
- Stone silos produce crystal shards
- Kitchens produce bones from meat meals
- Offering shrines generate webs, silks, and crystals

### Capacities
- Customise soul capacity multiplier
- Customise silo capacity multiplier
- Use multiples of 32 for silo capacity

### Auto-Collect
- Auto-loot from lumber/mine/farm/refinery chests when nearby
- Customise trigger amount and range
- Fast collection speed from shrines, beds, and chests

### Crafting Stations
- Halve refinery material requirements
- Mass fill refinery, cooking fire, kitchen, and pub queues with one click
- Eat food directly from follower kitchens (hold interact button)

### Weather
- Random weather changes on area exit
- Weather changes on day phase change
- Customise weather overlay colors
- Weather and phase change notifications

### Notifications
- Scarecrow trap catches
- Bed collapse warnings
- Low fuel warnings

### Unlocks
- Unlock Twitch drops and pre-order DLC
- Customise Sin Unlock boss amount

### Post Processing
- Remove vignette effect

## Installation

* Install [BepInExPack CultOfTheLamb](https://thunderstore.io/c/cult-of-the-lamb/p/BepInEx/BepInExPack_CultOfTheLamb/)
* Install [Configuration Manager](https://thunderstore.io/c/cult-of-the-lamb/p/p1xel8ted/BepInEx_Configuration_Manager/)
* Place the plugin DLLs into your "...\Cult of the Lamb\BepInEx\plugins" folder.

## Configuration

* Press F1 in game to open the configuration manager.
* The configuration file can be found in the "...\Cult of the Lamb\BepInEx\config" folder.

## Recommendation

While not required, I recommend installing [Unity Debugger Assistant](https://thunderstore.io/c/cult-of-the-lamb/p/LethalCompanyModding/UnityDebuggerAssistant/) for better log files when reporting bugs.
