# Glyph Override

Customize which controller button prompts are displayed in-game.

## Features

### Force Controller Prompts
- Override the auto-detected controller type to display specific button prompts
- Supported controller types:
  - Auto (default - uses connected controller)
  - Keyboard
  - DualShock 4 (PlayStation 4)
  - DualSense (PlayStation 5)
  - Xbox 360
  - Xbox One
  - Xbox Series
  - Nintendo Switch Joy-Cons (Detached)
  - Nintendo Switch Joy-Cons (Docked)
  - Nintendo Switch Handheld
  - Nintendo Switch Pro Controller

### Nintendo Switch A/B Button Fix
- Automatically swaps A/B button prompts when using any Nintendo Switch controller type
- Nintendo and Xbox controllers have physically swapped A/B positions:
  - Nintendo: A=right, B=bottom
  - Xbox: A=bottom, B=right
- The game already handles X/Y swap but not A/B - this mod adds the missing fix

## Installation

* Install [BepInExPack CultOfTheLamb](https://thunderstore.io/c/cult-of-the-lamb/p/BepInEx/BepInExPack_CultOfTheLamb/)
* Install [BepInEx Configuration Manager](https://thunderstore.io/c/cult-of-the-lamb/p/p1xel8ted/BepInEx_Configuration_Manager/) (for in-game configuration)
* Place the plugin DLL into your "...\Cult of the Lamb\BepInEx\plugins" folder.

## Configuration

* Press F1 in game to open the configuration manager.
* The configuration file can be found in the "...\Cult of the Lamb\BepInEx\config" folder.
