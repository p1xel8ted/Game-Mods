[size=4][b]Features[/b][/size]

[size=3][b]Co-op Controller Prompt Fix[/b][/size]
[list]
[*][size=3]Fixes Player 2 (Goat) seeing "--" instead of button prompts in local co-op.[/size]
[*][size=3]Button prompts now correctly update when Player 2 switches or reconnects their controller.[/size]
[*][size=3]Supports mixed controller types in co-op (e.g., Player 1 on Xbox, Player 2 on PlayStation).[/size]
[/list]

[size=3][b]Force Controller Prompts (Per-Player)[/b][/size]
[list]
[*][size=3]Override the auto-detected controller type to display specific button prompts.[/size]
[*][size=3]Separate settings for Player 1 (Lamb) and Player 2 (Goat) in co-op.[/size]
[*][size=3]Each player can be set to a different controller type independently.[/size]
[*][size=3]Useful when the game detects the wrong controller or when using adapters.[/size]
[*][size=3]Supported controller types:[/size]
[list]
[*][size=3]Auto (default - uses connected controller)[/size]
[*][size=3]Keyboard[/size]
[*][size=3]DualShock 4 (PlayStation 4)[/size]
[*][size=3]DualSense (PlayStation 5)[/size]
[*][size=3]Xbox 360[/size]
[*][size=3]Xbox One[/size]
[*][size=3]Xbox Series[/size]
[*][size=3]Nintendo Switch Joy-Cons (Detached)[/size]
[*][size=3]Nintendo Switch Joy-Cons (Docked)[/size]
[*][size=3]Nintendo Switch Handheld[/size]
[*][size=3]Nintendo Switch Pro Controller[/size]
[/list]
[/list]

[size=3][b]Nintendo Switch A/B Button Fix[/b][/size]
[list]
[*][size=3]Automatically swaps A/B button prompts when using any Nintendo Switch controller type.[/size]
[*][size=3]Works correctly per-player in co-op with mixed controller types.[/size]
[*][size=3]Nintendo and Xbox controllers have physically swapped A/B button positions:[/size]
[list]
[*][size=3]Nintendo: A=right, B=bottom[/size]
[*][size=3]Xbox: A=bottom, B=right[/size]
[/list]
[*][size=3]The game already handles X/Y swap but not A/B - this mod adds the missing fix.[/size]
[/list]

[size=4][b]Installation[/b][/size]

[list]
[*][size=3]Install [url=https://www.nexusmods.com/cultofthelamb/mods/31]Cult of the Lamb BepInEx 5 Pack[/url][/size]
[*][size=3]Install [url=https://thunderstore.io/c/cult-of-the-lamb/p/p1xel8ted/BepInEx_Configuration_Manager/]BepInEx Configuration Manager[/url] (for in-game configuration)[/size]
[*][size=3]Place the plugin DLL into your "...\Cult of the Lamb\BepInEx\plugins" folder.[/size]
[/list]

[size=4][b]Configuration[/b][/size]

[list]
[*][size=3]Press F1 in game to open the configuration manager.[/size]
[*][size=3]The configuration file can be found in the "...\Cult of the Lamb\BepInEx\config" folder.[/size]
[/list]
