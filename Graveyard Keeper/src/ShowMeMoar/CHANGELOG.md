# Changelog

## 0.1.11 | 18 April 2026

- Added an update notice on the main menu that flags when this mod (or others in this collection) has a newer version on Nexus. Click an entry to open its Nexus page. Toggle off in settings if you'd rather not see it

## 0.1.10 | 11 April 2026

- Removed unnecessary background processing that could affect performance
- Fixed hotkeys and input features not working for some users
- Improved compatibility across different BepInEx versions

## 0.1.9

- Added borderless windowed mode option
- Added ultrawide fog fix for displays wider than 2880px
- Mod is now standalone — no longer requires GYK Helper

## 0.1.6

- Option to toggle color correction
- Main menu artwork fixed. Looks fine on 21:9, not so sure about 32:9+. No change at 16:9.

## 0.1.5

- Minor change related to logging messages.
