[size=4][b]Features[/b][/size]

[size=3]This mod is experimental. Please back up your save files before using it![/size]

[list]
[*][size=3]Edit your character's appearance and name from the load game screen[/size]
[*][size=3]Click the edit button next to any saved character to reopen the character creator[/size]
[*][size=3]Backs up your save file automatically before making changes[/size]
[*][size=3]Open your save directory from the configuration menu[/size]
[/list]

[size=4][b]Configuration[/b][/size]

[size=3]All settings can be adjusted in-game by pressing F1 to open the configuration menu. This requires ConfigurationManager, which is included in the [url=https://www.nexusmods.com/sunhaven/mods/30]Sun Haven BepInEx Pack[/url].[/size]

[size=4][b]Installation[/b][/size]

[size=3]Install via Vortex (recommended), or manually place the mod DLL into your BepInEx/plugins folder.[/size]
